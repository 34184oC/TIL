# 1. CSS 선택자
  
<br>

## 1) 선택자(Selector)의 종류
CSS 선택자는 CSS 규칙을 적용할 요소를 정의한다.

<br>

### ✔ 주요 선택자
1. Type
```css
h2 {
  color: purple;
}
```
- 하나의 웹 페이지에서 일괄적으로 적용할 때 사용
- 대게 CSS 상단부에 작성

2. ID
```css
#welcome-title {
  color: red;
}
```
- 특정 요소 하나에만 적용되는 유일한 선택자
- CSS에서 태그 앞에 <pre><code>#</code></pre>을 붙임

3. Class
```css
.blue {
  color: blue;
}
```
- 스타일을 공통 적용해야 할 요소들을 그룹화 하기 위한 선택자
- 여러 요소에 중복 가능
- CSS에서 태그 앞에 <pre><code>.</code></pre>을 붙여 적용

<br>
<br>

### ✔ 속성 선택자
요소의 속성을 선택한다.
```css
/* [attr] */
a[target] {
  color: hotpink;
}

/* [attr=value] */
a[href="#"] {
  color: blue;
}
```
- input 태그는 type에 따라 여러 형태가 존재하기 때문에 속성 선택자를 활용
  ```css
  input[type="submit"] {
    background-color: green;
  }
  ```
